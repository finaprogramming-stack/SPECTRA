Мой стек технологий

Python

Мощный и гибкий язык, который я использую для создания сложной логики Telegram-ботов и серверной части сайтов. Идеально подходит для автоматизации и интеграций.

JavaScript

Оживляет сайты, делая их интерактивными и динамичными. Я использую его для создания анимаций, обработки действий пользователя и связи с сервером без перезагрузки страницы.

HTML5

Это каркас любого сайта. Я создаю на нем семантически правильную и структурированную разметку, которая является основой для дальнейшей стилизации и функциональности.

CSS3

Инструмент для визуального оформления сайта. С его помощью я создаю красивые и адаптивные интерфейсы, включая современные эффекты, такие как градиенты и анимации.

SQL

Язык для работы с базами данных. Он необходим для хранения и управления информацией, будь то данные о пользователях, товарах в интернет-магазине или история заказов.
